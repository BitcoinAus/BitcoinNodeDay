Why bitcoin node day?

Bitcoin nodes are the back bone of the bitcoin network and without them the network would not survive. Full nodes perform the following functions:

1. Filtering transactions and blocks on behalf of lightweight nodes so that lightweight nodes do not need to download every transaction ever made on the network in order to find their own transactions.

2. Serving historical full blocks to nodes that have been offline for a while. Transmitting new transactions from users to miners.

3. Broadcasting new blocks from miners to other nodes.

Get Involved on 30th August 2015 (~ block 372,765)

Inspired by Earth Hour, Bitcoin Node Day is designed to raise awareness around network node decline.

Bitcoin Node Day will be held on the 30th of August each year, starting at Midnight GMT. To participate, you can simply start bitcoin Core, buy or rent a full node.

What happens on Bitcoin Node Day?

Put simply, we are asking bitcoin enthusiasts to install and run a full bitcoin node, such as bitcoin core for 24hrs.

Visit the site or https://getaddr.bitnodes.io and see how the experiment works.

We also ask you to talk about bitcoin nodes, block sizes and the future of bitcoin with your peers, friends and family.

Run a desktop node

Bitcoin core is a desktop wallet that operates a full node. There are builds for Linux, Windows and Mac. Comprehensive instructions can be found at www.bitcoin.org

Configure your firewall / router

In order for other nodes to find and connect to yours you must allow a firewall rule on port 8333. After you have opened this port, you can check to see if the network can find your node by visiting https://getaddr.bitnodes.io
