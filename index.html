About the day

Inspired by Earth Hour, bitcoin node day is designed to raise awarness around bitcoin node decline. #bitcoinnodeday

What happens on bitcoin node day?

Put simple, we are asking bitcoin enthusists to install and run a full bitcoin node, such as bitcoin core for 24hrs.
